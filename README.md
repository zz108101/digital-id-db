# デジタルID制度 国際比較データベース

本リポジトリは、日本を基準に主要国の**デジタルID制度の思想・構造の違い**を  
一目で比較できることを目的とした、**データ駆動型の可視化サイト**です。

CSV（構造化データ）を唯一の正とし、  
Webサイトはその内容を自動的に表示します。

---

## 何ができるか

- 🇯🇵 日本 / 🇪🇪 エストニア / 🇰🇷 韓国 / 🇫🇷 フランス など  
  **複数国のデジタルID制度を横断比較**
- 住民登録・個人識別子・スマホID・mDL・認証基盤など  
  **同一指標・同一語彙で比較**
- CSVを更新するだけで  
  **Web表示が自動更新**
- 「published」データのみを表示するため  
  **未確定データが外に出ない設計**

---

## このサイトで分かること（例）

- なぜエストニアは「番号・連携先行型」なのか
- なぜフランスは「台帳を作らずにAPIで連携」しているのか
- 日本と韓国はどこが似ていて、どこが決定的に違うのか

👉 **制度の優劣ではなく、「設計思想の違い」を比較**することに重点を置いています。
